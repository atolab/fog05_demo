This demo shows the deployment of and MEC Applications and MEC Services using Eclipse fog05

Application instantiated using Eclipse fog05

  • MEC Platform
  • Wi-Fi Access Point
  • MQTT Broker
  • Wi-Fi RNIS (Radio Network Information Service)
  • RNIS Data Viewer - This web server

The goal of Eclipse fog05 is to address the heterogeneity that comes for having compute and networking resources at the edge of the network, because of the different virtualisation capabilities that will be present on lamp post or in a cabinet in the street, as well as the need of managing MEC applications as well as NSs in a scenario in which MEC and NFV should work together to enable the best experience for the user of the 5G network.
